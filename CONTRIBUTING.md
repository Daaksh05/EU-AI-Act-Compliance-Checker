# Contributing Guidelines

Thank you for your interest in contributing to the EU AI Act Compliance Checker.

## How to Contribute
1. Fork the repository
2. Create a new branch (`feature/your-feature-name`)
3. Make your changes
4. Commit with a clear message
5. Push the branch and open a Pull Request

## Code Style
- Follow PEP8 for Python code
- Add comments for regulatory logic
- Keep functions modular and readable

## Reporting Issues
If you find bugs or compliance mismatches, please open an issue with:
- Description
- Steps to reproduce
- Expected behavior

## Academic Use
This project can be used for educational and research purposes with attribution.
